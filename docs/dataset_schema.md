# Dataset Schema

This document summarizes the main columns generated by `scripts/collect_lqr_dataset.py`.

## Episode and Time

| Column | Meaning |
|---|---|
| episode | Episode index |
| control_step | LQR/control update step |
| physics_step | MuJoCo physics step |
| time_s | Simulation time |
| control_dt_s | Effective control timestep |

## Command Inputs

| Column | Meaning |
|---|---|
| speed_setpoint_m_s | Target forward velocity |
| yaw_setpoint_rad_s | Yaw command / wheel command offset |

## Ground Truth State

| Column | Meaning |
|---|---|
| x_m, y_m, z_m | Robot body position |
| quat_w, quat_x, quat_y, quat_z | Robot body quaternion in MuJoCo order |
| roll_rad, pitch_rad, yaw_rad | Euler angles |
| vx_world_m_s, vy_world_m_s, vz_world_m_s | World-frame linear velocity |
| vx_body_m_s, vy_body_m_s, vz_body_m_s | Body-frame linear velocity |
| wx_rad_s, wy_rad_s, wz_rad_s | Angular velocity |
| wheel_l_pos_rad, wheel_r_pos_rad | Wheel joint positions |
| wheel_l_vel_rad_s, wheel_r_vel_rad_s | Wheel angular velocities |
| encoder_forward_vel_true_m_s | Forward velocity estimated from wheel velocity |

## LQR Internal Values

| Column | Meaning |
|---|---|
| pitch_for_lqr_rad | Pitch value used by the LQR controller |
| pitch_dot_raw_rad_s | Raw pitch angular velocity |
| pitch_dot_filtered_rad_s | Filtered pitch angular velocity |
| wheel_vel_avg_raw_rad_s | Raw average wheel velocity |
| wheel_vel_avg_filtered_rad_s | Filtered average wheel velocity |
| velocity_linear_error_m_s | Target velocity error |
| lqr_motor_vel_raw_rad_s | Raw LQR motor velocity command |
| lqr_motor_vel_cmd_rad_s | Saturated motor velocity command |
| lqr_saturated | Whether motor command was saturated |
| ctrl_l_rad_s, ctrl_r_rad_s | Left and right actuator commands |

## Simulated Sensor Measurements

| Column | Meaning |
|---|---|
| gyro_x_meas_rad_s, gyro_y_meas_rad_s, gyro_z_meas_rad_s | Simulated gyro measurements |
| acc_x_meas_m_s2, acc_y_meas_m_s2, acc_z_meas_m_s2 | Simulated accelerometer measurements |
| enc_l_pos_meas_rad, enc_r_pos_meas_rad | Simulated encoder position measurements |
| enc_l_vel_meas_rad_s, enc_r_vel_meas_rad_s | Simulated encoder velocity measurements |
| enc_forward_vel_meas_m_s | Simulated encoder-based forward velocity |
| vision_x_meas_m, vision_y_meas_m, vision_z_meas_m | Simulated vision/position measurements |
| vision_yaw_meas_rad | Simulated vision yaw measurement |

## Learning Targets

| Column | Meaning |
|---|---|
| target_encoder_forward_vel_error_m_s | True forward velocity minus measured encoder forward velocity |
| target_gyro_x_error_rad_s | True gyro x minus measured gyro x |
| target_vision_x_error_m | True x minus measured vision x |
| target_vision_y_error_m | True y minus measured vision y |
| target_vision_yaw_error_rad | True yaw minus measured vision yaw |

## Intended Use

The target columns can be used to train an MLP residual estimator.

The trained model can later be inserted into the EKF pipeline as either:

1. measurement correction before EKF update, or
2. residual correction after EKF state estimation.
